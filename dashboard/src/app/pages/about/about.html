Interactive Microbiology

Interactive Microbiology Lab is an active research project at Riedel-Kruse Lab, Stanford University.

The goal of the project is to develop technologies to provide interactive access to a real biology lab over the Internet at scale for concurrent users. This involves designing and implementing custom hardware, Biotic Processing Units (BPU), software infrastructure with distributed computing and user friendly interfaces.

In this particular online lab, we provide a means to study the photo-taxis behavior of a single celled micro-organism called Euglena Gracilis through real experimentation over the Internet.

Users can run both live interactive or pre-programmed batch experiments through the website. Users can download the experiment data. Besides videos, the downloaded package contains light data, tracking data of every Euglena cell, and other meta data in Excel and JSON formats. Therefore, users need to only concentrate on experiment design - while running as many experiments as they want - without having to worry about any wetlab logistics.
